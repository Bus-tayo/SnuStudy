# 설스터디 MVP 해커톤 기획서

---

## 1. MVP 요건

### 개요
- **서비스 소개**  
  자체 콘텐츠 기반 수능 국영수 학습코칭 플랫폼

- **차별화 포인트**  
  - 공부 방법의 구조화  
    - OT / 예비문항을 통한 학생별 약점 파악  
    - 명확한 목표와 학습 본질 기반 맞춤형 공부 방법 제시  
  - 실행력 향상  
    - 하루 단위 체크리스트  
    - 매일 즉각적인 공부 피드백  
    - 꾸준한 학습 점검 및 질의응답  
  - 관리 품질의 표준화  
    - 과목별 정형화된 템플릿  
    - 학생의 공부 습관을 길러주는 자체 학습지

- **경쟁 제품**  
  에듀플렉스, 설탭, 김과외

- **개발 현황**  
  노션으로 앱과 유사한 구조 구현  
  (설스터디 노션 페이지 참고)

## 2. MVP 요건 정의서 (참가자 전달용)

### 개요
- 자체 콘텐츠 기반 수능 국영수 학습코칭 플랫폼
- 공부 방법 구조화 / 실행력 강화 / 관리 표준화
- 경쟁 서비스: 에듀플렉스, 설탭, 김과외
- 개발 현황: 노션 기반 MVP 구조 구현

---

### 필수 기능

- **일일 플래너 (멘티)**
  - 플래너 작성
  - 드래그 방식 공부 시간 체크
  - OCR 기반 자동 인식 (옵션)

- **인터랙티브 캘린더**
  - 주/월 단위 전환
  - 날짜 이동
  - 플래너/피드백 동기화

- **과제 업로드 및 관리**
  - PDF 자료 다운로드
  - 공부 인증 사진 업로드(jpg)

- **피드백 페이지**
  - 과목별 피드백 제공
  - 요약 피드백 강조
  - 과거 날짜 피드백 작성 가능

- **멘토 관리 화면**
  - 학생별 피드백 작성 집중 UI
  - 여러 학생 상태 빠른 파악
  - 피드백 즉시 노출 및 알림

---

### 추가 기능

- 알림: 과제 미완료 / 피드백 등록
- 채팅: 멘토·운영진과 1:1 소통
- 커뮤니티: 공부시간 비교 (열품타 형식)
- 게이미피케이션: 과제 달성률 기반 포인트 지급
